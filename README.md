# 金鼎盛金紙藝品商行

這是一個可直接部署到 GitHub Pages 的靜態網站。

## 上傳到 GitHub

把此資料夾內的檔案放到 GitHub repository 的根目錄：

- `index.html`
- `styles.css`
- `script.js`

GitHub Pages 會以 `index.html` 作為首頁。

## 作品照片

首頁已新增「作品展示」區塊。正式上線前可將每張作品卡的「上傳作品照片」改成實際照片與作品名稱。

目前第一個作品「一桶金」會讀取：

```txt
images/yitongjin.jpg
```

首頁已新增「下單詢問」表單。預設會開啟寄給 `hello@example.com` 的 Email 草稿；正式使用前請將 `index.html` 和 `script.js` 裡的 Email 改成店家的收件信箱。
